### Deployment:
1. Navigate to the simulation app directory
    ```
    cd simulator-app
    ```
2. Install and use the correct version of Node using <a href="https://github.com/nvm-sh/nvm">NVM</a>
    ```
    nvm install
    ```
3. Install all dependencies required
    ```
    npm install
    ```
4. Start the development server
    ```
    npm start
    ```
